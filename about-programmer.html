The Tools of the Trade

Rasmus is open-minded when it comes to what programming tools and operating systems that he must use when building applications. If he is able to choose himself though, he works in an Awesome-WM desktop environment, installed on a computer running Arch-Linux. He writes his code in Vim, an editor that will never stop to teach him new things. He doesn't neccessaily always consider the shell better than the GUI but he does find it the most invaluable tool in the toolbox of software that he utilizes to build applications.
